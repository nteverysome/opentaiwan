# OpenTaiwan AI Platform

Complete SaaS platform based on OpenHands with commercial features.

## Features
- User authentication with Supabase
- Quota management (10 free daily, 600 paid monthly)
- Modern UI with pricing and queue pages
- AI chat interface powered by OpenHands

## Setup
1. Install dependencies: `cd opentaiwan-api && npm install`
2. Configure environment variables in `.env.opentaiwan`
3. Run: `./start.sh`
